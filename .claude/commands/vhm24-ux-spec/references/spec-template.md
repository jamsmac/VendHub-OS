```

DELETE /api/v1/resourceName/:id
```

**Структура данных:**

```typescript
interface ResourceItem {
  id: number;
  name: string;
```

### 7. СОСТОЯНИЯ ЭКРАНА

**Loading:**

```
- Показать Skeleton для таблицы/карточек
- Disable кнопки действий
- Показать Spinner в заголовке
```

**Empty:**

```
- Показать EmptyState компонент
- Иконка: [IconName]
- Текст: "Нет данных"
- Действие: Кнопка "Добавить первый"
```

**Error:**

```
- Показать Alert variant="destructive"
- Текст: error.message
- Действие: Кнопка "Повторить"
```

**Success (после действия):**

```
- Toast уведомление
- Текст: "Успешно сохранено"
- Обновить список данных
```
