# Agent Skills / Competencies List

Ниже перечислены функциональные модули (скилы), доступные агенту в рамках архитектуры Cloud.MD.

## Skill: Навигатор Пространств (Space Navigator)
**Описание:** Способность двигаться от текущего состояния к целевому (гомеостатическому) в любом типе пространства.
**Принцип работы:**
- Определяет "цель" как координату в пространстве (например, правильная форма органа или смысловой вектор фразы).
- Минимизирует отклонение (ошибку) между текущим положением и целью.
**Примеры:**
- *Bio:* Клетки восстанавливают форму лица головастика даже при смещении исходных тканей ("Picasso tadpoles").
- *AI:* LLM выбирает следующее слово, минимизируя семантическое расхождение с контекстом.

## Skill: Адаптивный Ремаппинг (Adaptive Remapping)
**Описание:** Динамическое переписывание внутренних карт (эмбеддингов) при изменении контекста или повреждениях.
**Принцип работы:**
- Не использует жестко заданные карты. Обновляет скрытые представления (latent representations) на основе нового опыта.
- В ИИ реализуется через механизм внимания (Attention), который перевзвешивает важность токенов.
**Примеры:**
- *Bio:* Нейропластичность мозга; иммунная система создает новые антитела (VDJ-рекомбинация).
- *AI:* Few-shot learning (обучение на малом количестве примеров).

## Skill: Генеративное Шумоподавление (Generative Denoising)
**Описание:** Способность восстанавливать сложную структуру из хаоса (высокой энтропии) или поврежденных данных.
**Принцип работы:**
- Использует циклы "предсказание — коррекция". Процесс движется от шума к аттрактору (стабильному состоянию).
- Основан на термодинамике неравновесных процессов.
**Примеры:**
- *AI:* Диффузионные модели создают детальные изображения из случайного шума.
- *Bio:* Морфогенез (создание организма из шума делящихся клеток).

## Skill: Масштабируемая Координация (Scale-Free Coordination)
**Описание:** Способность решать задачи коллективно, где локальные действия агентов приводят к глобальному порядку.
**Принцип работы:**
- Агент (клетка/модуль) следует локальным правилам, но его среда настроена так, что это выгодно всей системе.
- Использует принципы Нейронных Клеточных Автоматов (NCA).
**Примеры:**
- Регенерация планарии (часть тела знает, как достроить целое).
- Роевой интеллект.

## Skill: Управление Критичностью (Criticality Regulation)
**Описание:** Умение переключаться между режимом стабильности (выполнение задач) и режимом пластичности (обучение/перестройка).
**Принцип работы:**
- Поддерживает систему в состоянии "самоорганизованной критичности".
- При сильном стрессе/повреждении система переходит в "жидкое" состояние для поиска нового решения (ремаппинг), затем кристаллизуется обратно для навигации.
