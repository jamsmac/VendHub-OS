# Прогнозная дозаправка — Карточка быстрого старта

**Распечатать и повесить на рабочем месте диспетчера**

---

## Утренний чек-лист (5 минут)

- [ ] Открыть `/dashboard/predictive-refill`
- [ ] Посмотреть красную карточку — **сколько срочных** (нужно закрыть сегодня)
- [ ] Нажать вкладку **"Срочно"**
- [ ] Проверить колонку **Тренд** — красные линии дозаправить в первую очередь
- [ ] Нажать **"Авто-маршрут"** — или отметить машины галочками и **"Добавить в маршрут"**
- [ ] Проверить маршрут (водитель, транспорт, время) и нажать "Запланировать"

---

## Быстрая расшифровка

| Бейдж                | Значение        | Действие                   |
| -------------------- | --------------- | -------------------------- |
| **Срочно** (красный) | < 2 дней запаса | Выезд сегодня              |
| **Скоро** (серый)    | 2–7 дней        | Планировать на этой неделе |
| **Норма** (серый)    | > 7 дней        | Ничего не делать           |

| Тренд         | Значение                              |
| ------------- | ------------------------------------- |
| Зелёная линия | Потребление стабильное                |
| Красная линия | Ускоряется — дозаправить раньше срока |

---

## После дозаправки

Найти строку в таблице → нажать кнопку **"Выполнено"**  
Без этого автомат останется в красной зоне до завтра.

---

## Если что-то не так

| Проблема                          | Решение                                        |
| --------------------------------- | ---------------------------------------------- |
| Таблица пустая                    | Данные ещё считаются — написать администратору |
| Цифры не обновились после продажи | Подождать 15–30 минут                          |
| Авто-маршрут не сработал          | Нет GPS у машин или нет срочных позиций        |
| Автомат красный после заправки    | Нажать **"Выполнено"** в таблице               |
